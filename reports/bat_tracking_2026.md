## Takeaways

- **Bat tracking carries forward — and it's about power, not contact.** Bat speed, blast/swing, and hard-swing rate predict *next-season* OPS/HR remarkably stably across both pairs (r ≈ 0.36–0.50); squared-up rate tracks AVG, not power, so it's dropped from the power-upside score. This is the rare bat-tracking metric set that actually repeats.
- **2026 batting buy-lows (elite tools, lagging output, still < 50% owned):** Smith, Cam, Mitchell, Garrett, Baty, Brett, McMahon, Ryan. These hitters' swings project more than the box score shows — regression-up candidates.
- **Velocity drives strikeouts, not run prevention.** In-season, fastball velo correlates ~0.46 with K/9 but only ≈ −0.14 with ERA — so 'stuff' buys you K9 (a category) far more reliably than ERA/WHIP. Spin is a weaker version of the same.
- **2026 SP stuff-vs-results buy-low watch:** Jones, Jared, Senga, Kodai, Pérez, Eury, Schultz, Noah. Big fastballs, results not there yet — but remember velo predicts K9 more than ERA, so expect the K9 to come before the ERA.
- **Sample caveat:** batting YoY rests on 2 full pairs; pitching is same-season only this round (predictive YoY deferred). `whiff_per_swing` is blank in the Savant export, so it carries no signal here.
